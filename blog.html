Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a way to organize a webpage's content as a tree of elements. It lets JavaScript change or update parts of the webpage without reloading it.

How do you select an element from the DOM?

To select an element from the DOM, use `document.getElementsByClassName('myClass')` to get elements by class name or `document.querySelector('div')` to select the first matching element. These methods enable easy access to and modification of HTML elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique that involves attaching a single event listener to a parent element instead of individual child elements. It is useful because it improves performance and simplifies code by allowing events to be handled even for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using `element.setAttribute('attributeName', 'value')` and get them with `element.getAttribute('attributeName')`. For styles, use `element.style.propertyName = 'value'` to change a specific CSS property directly.
